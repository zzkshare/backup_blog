发表于 2016-08-08   |   分类于 引导页   |     |   阅读次数

前言

介绍一下，我写博客的原因吧。真没想到，我也会突然想写文章。当然我想做博客也不是偶然，我从最近看的某教学视频某讲师那了解到了 markdown 这种神奇的标记语言； 同时被灌输了写一个自己的博客，有着种种好处（这里我就不灌鸡汤了）。但想搭建一个免费的个人博客又很不容易，从视频的讲师那我又了解到了 gitpages ,在看了几篇网上教程后成功搭建起我的博客(文章最后附上参考的教程)。

5.4 上传博客

当调试后觉的没问题了就可以上传到github了

等待完成后
可以去浏览器输入地址 https://你的用户名.github.io (这里https:// 不能少，不然打不开)

5.5

如果以上配置完成了，博客算是真正搭建完成了，还有一些主题的细节可以参考各自的主题说明文档（一般 github 里能找到）

6 开始写博客

  • 先对 Hexo 文件夹 选择 gitbash 然后 输入 hexo n “你的文件名”
  • 文章的位置在 Hexo\source_posts 目录中
  • 用支持 markdown 语法的编辑器（如 MarkdownPad， sublime）去编辑
  • 用 $ hexo s -g 调试
  • 调试完成用 $ hexo d -g 上传
